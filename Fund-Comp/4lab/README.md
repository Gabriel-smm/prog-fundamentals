Lab 4 Programs

Overview
- grades.c: computes the average and sample standard deviation of a fixed list of scores.
- ndfootball.c: menu-driven explorer for historical Notre Dame win/loss records (array-backed).
- primes.c: lists primes up to 1000 using the Sieve of Eratosthenes.

Build
- gcc grades.c -o grades -lm
- gcc ndfootball.c -o ndfootball -lm
- gcc primes.c -o primes -lm

Run
- ./grades
- ./ndfootball
- ./primes

Usage
- ndfootball shows a numbered menu for queries by year, minimum wins/losses, ranges, and combined filters; other programs print results directly.
