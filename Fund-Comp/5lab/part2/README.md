Lab 5 Part 2

Overview
- mysayings.c: stores up to 50 sayings, lets you list them, add new ones, search by substring, and save to a text file.

Build
- gcc mysayings.c -o mysayings

Run
- ./mysayings

Usage
- Start by entering the existing sayings file (e.g., a .txt file with one saying per line).
- Use the menu to list all sayings, add entries, search for a substring, or save to a new text file; choose option 5 to quit.
