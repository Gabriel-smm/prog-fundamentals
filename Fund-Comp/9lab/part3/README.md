Overview
- funanim.c animates a bouncing ball plus rotating squares and triangles in a gfx window; click to reposition and randomize the ball velocity.

Build
- make

Run
- ./funanim

Controls
- Left click: move ball to cursor and randomize velocity
- q: quit

Prerequisites
- Xming (or similar X11 server) must be running on Windows for the gfx graphics to display
