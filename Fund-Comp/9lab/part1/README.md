Lab 9 Part 1 — Symbol Drawer

Overview
- symbol.c opens a 500x300 gfx window and draws shapes at the cursor.

Build
- make


Run
- ./symbol

Controls
- Left click: blue square
- c: white circle
- t: green triangle
- 3–9: purple regular polygon with that many sides
- Esc: clear window
- q: quit

Prerequisites
- Xming (or similar X11 server) must be running on Windows for the gfx graphics to display
