Overview
- fractals.c renders multiple fractal drawings in a gfx window (triangles, squares, spirals, circles, snowflake, tree, fern, double spiral).

Build
- make

Run
- ./fractals

Controls
- 1: Sierpinski triangle
- 2: shrinking squares
- 3: spiral squares
- 4: circle snowflake
- 5: snowflake spokes
- 6: binary tree
- 7: fern
- 8: nested double spiral
- q: quit

Prerequisites
- Xming (or similar X11 server) must be running on Windows for the gfx graphics to display
