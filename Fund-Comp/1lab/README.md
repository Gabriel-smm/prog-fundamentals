Lab 1 Programs

Overview
- prog1.c: prompts for touchdowns, extra points, field goals, and safeties, then reports the total football score.
- prog2.c: computes the energy difference between two atomic levels of an electron using the Rydberg formula.

Build
- gcc prog1.c -o prog1
- gcc prog2.c -o prog2 -lm

Run
- ./prog1
- ./prog2

Usage
- Follow the on-screen prompts for the required numeric inputs.
